# Simple Academic Homepage Template 6

A classic academic layout with traditional header design and organized sections.

## Features

- Traditional header with photo
- Clean section organization
- Timeline-style education and experience
- Professional typography
- Fully responsive

## Perfect For

- Academic researchers
- Graduate students
- Professors
- Personal academic websites

## Customization

### Adjust Header Layout

```css
.header-content {
    display: flex;
    gap: 40px;  /* Space between text and photo */
}
```

### Modify Colors

```css
.section-title {
    border-bottom: 2px solid #e0e0e0;  /* Section divider */
}
```
