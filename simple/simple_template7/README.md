# Simple Academic Homepage Template 7

A card-based compact design with clean card layouts for each section.

## Features

- Card-based layout
- Compact header design
- Interest tags display
- Clean card styling
- Fully responsive

## Perfect For

- Academic researchers
- Graduate students
- Professors
- Personal academic websites

## Customization

### Change Card Styling

```css
.info-card {
    border: 1px solid #e0e0e0;  /* Card border */
    border-radius: 6px;  /* Card corners */
}
```

### Modify Colors

```css
body {
    background: #f5f5f5;  /* Page background */
}

.info-card {
    background: #ffffff;  /* Card background */
}
```
