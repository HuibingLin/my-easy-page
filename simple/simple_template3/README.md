# Simple Minimalist Homepage Template 3

A minimalist design with dark color accents and clean typography.

## Features

- Dark accent color scheme (black/dark gray)
- Tag-based research interests display
- Timeline-style layout for education and experience
- Clean publication list with links
- Fully responsive

## Perfect For

- Personal academic websites
- Professional portfolios
- Research profiles
- Minimal landing pages

## Customization

### Change Primary Color

```css
.header {
    border-bottom: 2px solid #2a2a2a;  /* Change this */
}

.tag {
    background: #2a2a2a;  /* Change tag background */
}

.timeline-marker {
    background: #2a2a2a;  /* Change marker color */
}
```

### Popular Color Options

- Navy: `#1a365d`
- Forest Green: `#22543d`
- Burgundy: `#742a2a`
- Charcoal: `#2a2a2a`
