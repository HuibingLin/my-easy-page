# Simple Minimalist Homepage Template 4

A minimalist design with warm earth tone accents and card-based layout.

## Features

- Warm color scheme (terracotta/earth tones)
- Card-based content layout
- Info bar with highlighted contact details
- Clean and inviting design
- Fully responsive

## Perfect For

- Personal websites
- Academic profiles
- Professional portfolios
- Creative professionals

## Customization

### Change Accent Color

```css
.avatar {
    border: 3px solid #c87941;  /* Change border color */
}

.section-heading {
    color: #c87941;  /* Change heading color */
}

.card {
    border-left: 3px solid #e8d5c4;  /* Change card accent */
}

.card:hover {
    border-left-color: #c87941;  /* Change hover color */
}
```

### Popular Warm Color Options

- Terracotta: `#c87941` (default)
- Coral: `#f18f7f`
- Burnt Orange: `#cc5500`
- Warm Brown: `#8b6f47`
- Rust: `#b7410e`

### Change Background Color

```css
body {
    background: #fefdf8;  /* Warm off-white */
    /* Or try: #fef9f3, #fdf8f0 */
}
```
