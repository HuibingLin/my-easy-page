# Simple Academic Homepage Template 5

A minimalist design with left sidebar navigation, inspired by classic academic websites.

## Features

- Left sidebar with navigation menu
- Profile photo and basic info in sidebar
- Smooth scroll navigation
- Clean main content area
- Fully responsive

## Perfect For

- Academic researchers
- Graduate students
- Professors
- Personal academic websites

## Customization

### Change Sidebar Width

```css
.sidebar {
    width: 250px;  /* Adjust sidebar width */
}
```

### Modify Colors

```css
.sidebar {
    background: #f8f9fa;  /* Sidebar background */
}

.nav-link.active {
    background: #333;  /* Active nav color */
}
```
