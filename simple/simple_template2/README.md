## How to Use

### 1. Download Files

Download `index.html` and `style.css` to your computer.

### 2. Add Your Photo

Replace the placeholder image:

```html
<img src="your-photo.jpg" alt="Your Photo" class="profile-photo">
```

### 3. Edit Your Information

Update in `index.html`:

- **Name**: Replace "Your Name"
- **Subtitle**: Your professional title
- **Contact**: Email address
- **Social Links**: GitHub, Scholar, LinkedIn
- **About**: Brief introduction
- **Interests**: Your research/work interests
- **Education**: Educational background
- **Publications**: Your papers
- **Experience**: Work experience

### 4. Customize Styling

Open `style.css` to adjust:

- Font sizes and weights
- Colors
- Spacing
- Layout width

### 5. Update Publications

Add your publications:

```html
<div class="item">
    <span class="item-title">Paper Title</span>
    <span class="item-meta">Conference/Journal, Year</span>
    <div class="item-links">
        <a href="#" target="_blank">Paper</a>
        <a href="#" target="_blank">Code</a>
    </div>
</div>
```

## Customization Tips

### Change Background Color

```css
body {
    background: #fafafa;  /* Light gray */
    /* Or try: #ffffff for white */
}
```

### Adjust Container Width

```css
.container {
    max-width: 600px;  /* Adjust for wider/narrower layout */
}
```

### Modify Typography

```css
.name {
    font-size: 2.4rem;  /* Header name size */
    font-weight: 300;   /* Lighter weight */
}

.section-text {
    font-size: 1rem;    /* Body text size */
    font-weight: 300;   /* Lighter weight */
}
```

### Change Colors

```css
.name {
    color: #1a1a1a;  /* Dark text */
}

.subtitle {
    color: #666;     /* Gray text */
}
```

### Adjust Spacing

```css
.content-section {
    margin-bottom: 50px;  /* Space between sections */
}

.profile-section {
    margin-bottom: 60px;  /* Space after profile */
}
```
